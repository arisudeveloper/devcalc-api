# TP1 - Pipelines de CI/CD e DevOps 


Este projeto tem como objetivo praticar conceitos de CI/CD com GitHub Actions.

## Ferramenta de Build
Utilizei o Maven para gerenciar as dependências ( Javalin, JUnit) e automatizar o processo de build.

## Como rodar a aplicação
1. Certifique-se de ter o Java 21 e o Maven instalados.
2. No terminal, execute:
   mvn clean install
